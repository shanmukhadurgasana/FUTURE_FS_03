# Crafty Heaven Static Website

This project is now implemented as a static single-page website with only three source files:

- index.html
- main.js
- style.css

## Run

1. Install dependencies:

npm install

2. Start development server:

npm run dev

3. Build for production:

npm run build

4. Preview production build:

npm run preview

## Notes

- Routing is handled in main.js using the browser History API.
- Cart and wishlist are in-memory state for this demo.
- The older React/TypeScript source folder is no longer required for this static implementation.
